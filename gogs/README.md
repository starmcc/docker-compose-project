# Gogs 极易搭建Git私有仓库

## 前言

极易搭建和简洁的界面是该服务的特点，相比于Gitlab，使用Go开发的Gogs更加的轻量级。

适用于个人或小团体进行Git的私有仓库部署。



## 步骤

**Step.1**

将同级目录下的`docker-compose.yml`复制到服务器的自定义目录中。

**Step.2**

使用`docker-compose up -d`启动容器。

**Step.3**

浏览器输入`xxx.xxx.xxx:3000`进入初始化设置界面(**底部可选面板语言为中文**)。**注意端口开放！**

**Step.4**

设置成功后刷新页面即部署成功！
